I also have a web feed set up if you prefer to read your news in an aggregator of some sort. Both methods have the same content, the only difference is the delivery method. You don't actually need to click that link (the file is there for the machines to read), you can just search/add http://pawnstorm.net or http://pawnstorm.net/atom.xml (if the first one doesn't work for some reason) in your feed reader of choice and you'll be good to go.

If you want to know more about web feeds, I would suggest checking out aboutfeeds.com, which is a site put together by Matt Webb to help folks who are new to web feeds. Personally, it is my preferred way of interfacing with the web (it's kind of like most of the things that I value about social media without all the stuff that bothers me).

If you are looking for a feed reader, there are plenty to choose from, but I'm a fan of NewsBlur myself (you can see what I'm reading there, if you want).
